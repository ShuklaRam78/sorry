# Sorry ❤️

A heartfelt apology web page with animated hearts, a playful "Will you forgive me?" prompt, and a beautiful apology message.

## Features
- Animated falling hearts on the main page
- Big "SORRYYYYY" title with a "Will you forgive me?" prompt
- Yes/No buttons — the "No" button runs away from the cursor!
- Clicking "Yes" reveals a lovely apology page with raining hearts and a sincere message

## How to Run
1. Open a terminal in the `src` folder
2. Run: `python3 -m http.server 8080`
3. Open [http://localhost:8080](http://localhost:8080) in your browser

## Files
- `src/index.html` — Main HTML page
- `src/style.css` — Styles
- `src/main.js` — Heart animations, evasive No button, apology page logic
